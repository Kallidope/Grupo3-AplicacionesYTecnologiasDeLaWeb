# Grupo3-AplicacionesYTecnologiasDeLaWeb
Repositorio para agregar los Laboratorios y Evaluaciones del Grupo 3 de la asignatura Aplicaciones y Tecnologías de la Web.
Este espacio reúne los laboratorios, evaluaciones y avances del semestre, organizados de forma clara para facilitar su desarrollo y revisión.

## Objetivo
Organizar y respaldar los trabajos desarrollados en la asignatura, manteniendo un registro ordenado de cada avance y versión del proyecto.

## Consideración
El repositorio está estructurado por carpetas y versiones para facilitar la organización, el trabajo colaborativo y la revisión del material.
