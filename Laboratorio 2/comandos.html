Diccionario de Operaciones

  • docker build -t nombre-imagen . : Construye una imagen a partir del Dockerfile en la carpeta actual.
  • docker run -d -p 8080:80 nombre-imagen : Ejecuta un contenedor en segundo plano mapeando el puerto 8080.
  • docker ps : Muestra los contenedores que están corriendo actualmente.
  • docker images : Lista todas las imágenes disponibles localmente.
  • docker stop [ID_o_Nombre] : Detiene un contenedor en ejecución.
  • docker rm -f [ID_o_Nombre] : Elimina un contenedor de forma forzada.
