Proceso de instalación

  1. Descargar Docker Desktop desde el sitio oficial.
  2. Ejecutar el instalador y aceptar la configuración recomendada.
  3. Reiniciar el equipo si el sistema lo solicita.
  4. Abrir Docker Desktop y esperar a que el estado sea correcto.

Prueba inicial

Para comprobar que Docker quedó instalado correctamente, se utiliza el siguiente comando:

docker run hello-world

Este comando descarga una imagen de prueba y verifica que Docker esté funcionando.

Construcción y ejecución del sitio

Una vez creado el sitio HTML, se construye la imagen del proyecto con:

docker build -t mi-web-html .

Luego se ejecuta el contenedor con:

docker run -d -p 8080:80 --name mi-web-container mi-web-html

Finalmente, el sitio se visualiza en el navegador ingresando a:

http://localhost:8080
