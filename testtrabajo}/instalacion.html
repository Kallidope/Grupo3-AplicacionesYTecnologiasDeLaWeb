Proceso de Configuración en Windows

  1. Descarga: Obtener el instalador de Docker Desktop desde el sitio oficial.
  2. WSL 2: Asegurarse de tener habilitado el Subsistema de Windows para Linux (WSL 2) en las características del sistema.
  3. Instalación: Ejecutar el .exe y seguir el asistente. Es necesario reiniciar la computadora.
  4. Ejecución: Abrir la aplicación Docker Desktop y esperar a que el icono de la ballena esté en verde.
